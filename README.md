## 🚀 Descripción

Aplicación web construida con **React + TypeScript + Vite** que consume la API backend para la gestión de productos, categorías, ingredientes, pedidos, pagos y usuarios.

---

## ✨ Funcionalidades

- Autenticación de usuarios
- Gestión de productos, categorías e ingredientes
- Carrito de compras
- Procesamiento de pedidos y pagos
- Dashboard con estadísticas
- Panel de administración

---

## 📁 Estructura del proyecto

```
src/
├── features/   # auth, cart, categories, ingredients, orders, payment, products, user
├── pages/      # Landing, Dashboard, Products, Categories, etc.
├── shared/     # api, components, constants, helpers, hooks, layouts, theme
├── widgets/    # header, sidebar
├── App.tsx
├── index.css
└── main.tsx
```
